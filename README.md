# Study English with AI!

An interactive language learning platform powered by AI for mastering English through comprehensive practice modules.

## 🌟 Features

- **📚 Vocabulary**: Interactive flashcards for memorizing Korean-English translations
- **✍️ Writing**: Practice translations with AI-powered evaluation and feedback
- **🗣️ Speaking**: Simulated conversations with AI for speaking practice
- **👂 Listening**: Audio comprehension exercises with questions
- **📖 Reading**: Image-to-text parsing with automatic translation

## 🚀 Live Demo

Visit: [https://s-choung.github.io/english-ai-study/](https://s-choung.github.io/english-ai-study/)

## 🔧 Setup Instructions

1. Fork or clone this repository
2. Visit the deployed site
3. Click the settings button (⚙️) in the top-right corner
4. Enter your OpenAI API key
5. Start learning!

## 🔐 Security

- API keys are stored only in browser session storage
- Keys are never committed to the repository
- Each user provides their own API key
- Sessions clear when the browser is closed

## 💻 Technologies Used

- HTML5/CSS3
- Vanilla JavaScript
- OpenAI API (GPT-4/GPT-3.5)
- GitHub Pages for hosting
